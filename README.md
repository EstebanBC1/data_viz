# data_viz

A data visualization repository focused on making data exploration and charting easier and faster.

## About

This project is building toward an **MVP (Minimum Viable Product)** that assists with common `pandas` workflows and data visualization tasks. The goal is to reduce the boilerplate involved in exploring, cleaning, and visualizing data so you can go from a raw dataset to useful charts and insights with less friction.

Planned areas of focus include:

- **Pandas assistance** — helpers for common data wrangling tasks (loading, cleaning, filtering, aggregating).
- **Data visualization** — streamlined ways to generate charts and plots from `pandas` DataFrames.
- **Ease of use** — a simple, intuitive interface aimed at both quick exploration and repeatable workflows.

This project is under active development. Features, structure, and documentation will evolve as the MVP takes shape.
